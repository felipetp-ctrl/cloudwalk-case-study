# Edge Inference Benchmark

Benchmark tool for measuring LightGBM (ONNX) inference latency in native Rust and WASM. Supports both tiers of the detection system.

## Prerequisites

- Rust toolchain (`rustup`)
- `wasmtime` CLI (`brew install wasmtime`)
- WASM target: `rustup target add wasm32-wasip1`
- ONNX model files in `model/` (generated by `notebooks/edge_deployment.ipynb`)

## Models

| Model | File | Features | Description |
|-------|------|----------|-------------|
| Tier 1 (request-level) | `model/model_full.onnx` | 36 | Per-request classification |
| Tier 2 (source-level) | `model/source_model.onnx` | 13 | IP-day behavioral classification (DDoS) |

## Usage

### Native benchmark

```sh
cargo build --release

# Tier 1 — request-level model (default)
cargo run --release

# Tier 2 — source-level model
cargo run --release -- model/source_model.onnx model/source_sample_input.json
```

### WASM benchmark

```sh
cargo build --release --target wasm32-wasip1

# Tier 1
wasmtime run --dir model target/wasm32-wasip1/release/edge-inference.wasm

# Tier 2
wasmtime run --dir model target/wasm32-wasip1/release/edge-inference.wasm -- model/source_model.onnx model/source_sample_input.json
```

### Custom arguments

```sh
cargo run --release -- <model_path> <sample_input_path> <n_iterations>
```
